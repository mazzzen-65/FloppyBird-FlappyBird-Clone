# Floppy Bird

A Flappy Bird clone written in Java.

---

## How to Run

1. **Clone the repository** or **[download the source code](https://github.com/mazzzen-65/FloppyBird-FlappyBird-Clone/releases/tag/v1.0.0)**.  
2. Open the project in your IDE (IntelliJ, Eclipse, or any Java IDE).  
3. Run `App.java` to start the game.
